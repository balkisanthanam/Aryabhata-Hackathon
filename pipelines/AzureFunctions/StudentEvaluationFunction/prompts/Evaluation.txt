# EVALUATION INSTRUCTIONS

## 1. Locate Each Problem
- Scan ALL student pages to find the work for each listed problem
- Students often label their work (e.g., "Q13.9", "13.9", "Problem 13.9") — look for these markers
- If a problem's work spans multiple pages, consider ALL parts together
- **"Not Found" means ZERO work exists under that problem label.** Only use "Not Found" when the student has genuinely not written anything for that problem number.

### CRITICAL: Work Present but Wrong Problem Solved
If the student HAS work labeled with the problem number, but the work answers a DIFFERENT question than the assigned textbook problem:
- This is **NOT** "Not Found" — the student DID submit work under that label
- Mark `found_in_student_work: true`
- Set `evaluation_status: "Answer is Incorrect"`
- Set `error_type: "misread_problem"` in `error_pinpoint`
- In `student_wrote`: describe what the student actually solved
- In `expected`: describe what the assigned problem actually asked
- This distinction is essential — the student needs to know they solved the wrong problem, not that their work is missing

## 2. Answer Correctness
- Use the chapter PDF to find the problem statement and derive the correct answer
- If no PDF: solve the problem yourself using your expertise
- Accept answers within reasonable error margins (decimal precision, rounding, significant figures)
- For numerical answers: accept if within ±1% or appropriate precision for the problem level
- For algebraic/equation answers: accept equivalent forms (e.g., x=5 and 5=x)

## 3. Solution Analysis — PINPOINT THE ERROR
If answer is INCORRECT or ACCEPTABLE (partially correct):

**You MUST identify the EXACT point of divergence:**
- State the exact step number where the student went wrong (e.g., "In Step 3 of your solution...")
- Quote what the student actually wrote vs. what was expected
- Classify the error type precisely:
  - `conceptual_misconception` — Wrong understanding of the principle/formula/law
  - `wrong_formula` — Used an incorrect or inapplicable formula
  - `sign_error` — Positive/negative sign mistake
  - `unit_error` — Wrong units or failed unit conversion
  - `algebraic_error` — Mistake in algebraic manipulation
  - `arithmetic_error` — Simple calculation mistake (addition, multiplication, etc.)
  - `incomplete_solution` — Correct approach but stopped before reaching the final answer
  - `misread_problem` — Misinterpreted the problem statement or diagram
- Indicate **severity**: "fundamental" (needs to re-study the concept) or "minor" (careless slip, just needs more practice/care)

If answer is CORRECT:
- Note if all steps are shown clearly
- Mention if the method is optimal or could be simplified
- Note presentation quality

## 4. Feedback & Teaching
- Reference specific sections from the PDF chapter when explaining concepts
- In the student tip: be specific and actionable (e.g., "Re-read Section 12.3 on Le Chatelier's Principle" not just "Review the chapter")
- For INCORRECT answers: the tip should address the specific error type found
- For CORRECT answers: give encouraging feedback and optionally suggest efficiency improvements
- For diagrams/figures in problem: ensure student interpreted them correctly

## 5. Complete Solution
- Provide step-by-step solution with clear explanations
- Show all intermediate steps
- Highlight key concepts and formulas used
- Match the solution complexity to {class}th standard level

## 6. LaTeX Formatting (MANDATORY)
- Format ALL mathematical expressions using LaTeX: `$...$` for inline, `$$...$$` for display/block
- Format ALL chemical formulas using `$\ce{...}$` (e.g., `$\ce{CH3COOH}$`, `$\ce{H2SO4}$`, `$\ce{NaOH + HCl -> NaCl + H2O}$`)
- Format ALL chemical reactions with proper arrow notation: `$\ce{->}$` for forward, `$\ce{<=>}$` for equilibrium
- Never write raw chemical formulas as plain text (e.g., write `$\ce{CH3COOH}$` not `CH3COOH`)
- Use `$\ce{...}$` for IUPAC names that contain chemical notation
- Fractions: use `$\frac{a}{b}$` not `a/b`
- Superscripts/subscripts: use `$x^2$`, `$H_2O$` not `x2`, `H2O`
