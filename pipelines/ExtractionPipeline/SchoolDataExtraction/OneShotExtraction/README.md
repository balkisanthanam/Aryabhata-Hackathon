# PDF Extraction Pipeline

This folder contains scripts for extracting questions and images from PDF files (primarily CBSE exam papers).

## Main Files
- `ExtractImagesFromPDF.py` - Core PDF extraction logic that loops through PDFs in input folder
- `run_extraction.py` - Runner script for CBSE PDF extraction
- `config.json` - Configuration for PDF extraction process

## Usage
Run the extraction on PDFs in the input folder using the runner script.
