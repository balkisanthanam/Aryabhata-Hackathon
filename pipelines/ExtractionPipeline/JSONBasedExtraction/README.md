# JSON-Based Extraction Pipeline

This folder contains scripts for extracting structured question data from educational PDFs (JEE papers, CBSE textbooks, etc.) as JSON.

## Main Files
- `run_json_extraction.py` - Main script to extract questions from PDFs as structured JSON
- `evaluate_extraction.py` - Evaluation tool to compare extraction results against model outputs

## Usage

Run the JSON extraction script to process PDFs and extract structured question data.

```powershell
python run_json_extraction.py
```
